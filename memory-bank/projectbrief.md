## Обзор
**Skazmor Playtest** — это техническая утилита (sandbox) и веб-приложение для прототипирования и балансировки игровой механики будущей игры в жанре **Roguelike Solitaire** (боевой ролевой пасьянс, референс: *Card Crawl*).

Проект служит рабочей средой для Геймдизайнера и Разработчика, позволяя проверять гипотезы, тестировать новые карты и настраивать экономику игры без необходимости запускать полноценный продакшн-билд игры (которая может быть разработана отдельно).

## Ключевые цели
1.  **Инструмент балансировки:** Предоставить Геймдизайнеру гибкий интерфейс для проверки математики игры (урон, здоровье, стоимость карт).
2.  **Песочница механик (V2):** Создать расширяемую архитектуру для тестирования сложных сценариев (разные "комнаты", новые типы врагов, режимы игры).
3.  **Доступность:** Быстрое развертывание обновлений через Vercel (skazmor.app) для удаленного тестирования.

## Статус проекта
*   **Текущая стадия:** V1 завершена (базовый цикл боя работает).
*   **В работе:** Переход к V2 — расширение контента, улучшение архитектуры для масштабируемости, добавление новых механик.
*   **Мета-слой:** Существует "Dev Log Quest" (консоль разработчика) — развлекательный модуль для команды, не являющийся частью основного продукта.
